
# Maze Hunter

this project is a procedurally generated maze game You must navigate through a massive maze while being hunted by a monster that uses A* pathfinding to track you down.




## Features

- Procedurally Generated Maze 
- Fog of War
- Smart Monster AI
- Visual Enhancements
- Customizable Settings
- Main Menu


## Controls

| Action | Key                |
| :-------- | :------------------------- |
| `Move Up` | W / ↑ |
| `Move Down` | S / ↓ |
| `Move Left` |  A / ←|
| `Move Right` | D / → |
| `Exit Game` | ESC |


> [!NOTE]
> This project has multiple versions make sure to download the latest release.


## Installation

#### 1. Install Dependencies

 Make sure you have Python 3.10+ installed.

```bash
  pip install pygame
```
#### 2. Git clone This Repo

```sh
https://github.com/for-tristan/Maze-hunter
```

#### 3. Run the Game
You can start the menu by running:

```bash
python menu.py
```
#### Or
```bash
python Game.py
```
